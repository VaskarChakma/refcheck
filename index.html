How it works
Step 01

Extract

PDF text is parsed in your browser via PDF.js. The reference section is located automatically and each citation is isolated regardless of formatting style.

Step 02

Query

Each reference is searched across CrossRef, Semantic Scholar, and OpenAlex simultaneously. All three APIs are completely free and require no account or key.

Step 03

Match

Title keywords, author names, publication year, and DOIs are compared against search results using a weighted similarity score.

Step 04

Report

Results appear as each reference resolves. Export a full CSV or JSON report suitable for editorial records or peer review documentation.
